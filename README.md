Play .MKV Files In Mario Kart Style!

MarioKartVideo is a custom desktop launcher built with Python and PyQt6 designed to provide a nostalgic console experience while navigating media files. This application recreates the Mario Kart Wii title screen environment, featuring authentic sound transitions and the iconic Wii pointer interface.

Features
Wii-Style Interface: A full-screen experience utilizing high-resolution assets.

Custom Wii Cursor: Replaces the standard Windows arrow with the classic Wii hand pointer (WIICURSOR.png).

Audio-Driven Transitions: Screen transitions occur after the selection sound (Select.mp3) finishes, mimicking console behavior.

Authentic BGM: Loops the Mario Kart Wii title theme (TTBGM.mp3) seamlessly in the background.

Custom Quit Dialog: An Escape key menu with styled buttons to confirm application exit.

Optimized for Modern Python: Developed for compatibility with Python 3.13 and above.
